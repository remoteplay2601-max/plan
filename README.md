# Streamlit - Qualifab Genius Input

## Demarrage rapide
1) Installer les dependances: `pip install -r requirements.txt`
2) Lancer l'app: `streamlit run app.py`
3) Choisir un mode, charger un fichier Excel, puis renseigner les valeurs.

## Notes
- L'app tente d'ouvrir un dialogue Windows (tkinter) pour choisir les chemins locaux.
- Si le dialogue est indisponible, renseignez un chemin de sauvegarde manuel.
